# 🏡 Roamly

Roamly is a full-stack accommodation listing platform inspired by Airbnb, where users can explore destinations, create listings, upload images, leave reviews, and discover places on an interactive map.

## 🌐 Live Demo

https://roamly-3gnt.onrender.com

## 📸 Features

* User Authentication & Authorization
* Create, Edit, and Delete Listings
* Upload Images using Cloudinary
* Interactive Maps using Mapbox
* Reviews & Ratings System
* Responsive Design
* Secure Session Management
* MongoDB Atlas Cloud Database

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/JaganRamadugu/Roamly.git
```

Navigate to project directory:

```bash
cd Roamly
```

Install dependencies:

```bash
npm install
```

Create a `.env` file and add:

```env
ATLASDB_URL=your_mongodb_connection_string

CLOUD_NAME=your_cloudinary_cloud_name
CLOUD_API_KEY=your_cloudinary_api_key
CLOUD_API_SECRET=your_cloudinary_api_secret

MAP_TOKEN=your_mapbox_token
```

Run the application:

```bash
node app.js
```

Visit:

```plaintext
http://localhost:1010
```
